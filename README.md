## Overview

Melodyne Studio is aimed at precision audio editing in desktop music production. It centers on pitch tuning, timing refinement, formant adjustment, and note-level control, giving producers, engineers, and sound designers detailed access to recorded audio.

It also works well in multi-track and DAW-based setups thanks to ARA2 integration. With MIDI export for melodic content and plugin support through VST3, AU, and AAX, it can slot into many studio environments without forcing a different workflow.

## What it Offers

- Direct Note Access for note-by-note editing of polyphonic audio
- Tools for pitch correction, timing correction, and formant shifting
- ARA2 integration for smoother operation in supported DAWs
- MIDI export to convert melodies into editable MIDI material
- Multi-track audio editing for layered production workflows
- Multi-threaded DNA engine for efficient analysis and editing
- VST3, AU, and AAX support for wide plugin compatibility
- Sound design-focused controls for detailed vocal and instrument shaping

## Usage

A common workflow looks like this:

1. Import an audio file or open a session in your DAW.
2. Let Melodyne analyze the material.
3. Select notes, phrases, or tracks for editing.
4. Adjust pitch, timing, and formant behavior as needed.
5. Use Direct Note Access for polyphonic passages when supported.
6. Export melody content to MIDI when you want to continue arrangement work in another tool.

For DAW-based work, load the plugin in a compatible host and enable ARA2 where your setup supports it.

## Configuration

Most controls are handled inside the application and through your DAW project settings.

Places worth checking include:
- application preferences for analysis and editing behavior
- plugin settings inside your host for format and integration options
- project files for session-specific edits and recall

If integration is not behaving as expected, confirm that the proper plugin format is installed and that your DAW supports ARA2 for the workflow you plan to use.

## FAQ

**How do I get updates?**  
Use the download link above to check the latest release build and compare it with the version shown in the badge.

**Why is my DAW not showing the plugin?**  
Make sure the correct plugin format is installed, then refresh your DAW's plugin scan and review any host-specific plugin paths.

**Where are my settings stored?**  
Most preferences are kept in the app itself or in your DAW project/session files, depending on how you are using the software.

**What if analysis or playback feels slow?**  
Close projects you are not using, reduce background load, and confirm that your system has enough CPU, memory, and disk performance for multi-track editing.

**Can I use it outside a DAW?**  
Yes, the desktop app supports standalone audio editing workflows, while ARA2 and plugin formats are available for host-based use.
